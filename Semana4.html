SEMANA 4

Exposiciones Parte 1

Resumen de la Semana


Esta semana fue la entrega de las exposiciones delegadas por el ingeniero a cargo del curso, los temas tratados fueron los siguientes: HTML básico, HTML avanzado, CSS Básico, CSS Avanzado, Temas de Bootstrap y Temas de Tailwind CSS

HTML Basico

En esta exposicion se introdujo el lenguaje de marcado HTML (HyperText Markup Language), el cual es la base de toda página web. Aprendimos a estructurar contenido utilizando etiquetas básicas que definen encabezados, párrafos, listas, enlaces, imágenes y tablas. También se explicó la anatomía de una página HTML.

Contenido Aprendido

  • Estructura básica: <!DOCTYPE html>, <html>, <head>, <body>
  • Encabezados y párrafos: uso de <h1> a <h6> y <p>
  • Listas: ordenadas <ol> y no ordenadas <ul> con <li>
  • Enlaces e imágenes: <a href=""> y <img src="">
  • Comentarios: <!-- Comentario -->

HTML Avanzado

En esta exposicion profundizamos en las etiquetas más avanzadas de HTML. Aprendimos a crear formularios interactivos, incrustar contenido multimedia y utilizar tablas para organizar datos.

Contenido Aprendido

  • Formularios: <form>, <input>, <textarea>, <select>, <button>
  • Tablas: <table>, <tr>, <td>, <th>, <thead>, <tbody>
  • Multimedia: <audio>, <video>, <iframe>
  • Semántica avanzada: <section>, <article>, <aside>, <figure>, <figcaption>

CSS Basico

Durante esta exposicion exploramos los fundamentos de CSS (Cascading Style Sheets), aprendiendo cómo aplicar estilos a nuestras páginas HTML para mejorar su apariencia visual y experiencia de usuario.

Contenido Aprendido

  • Sintaxis de CSS: selectores, propiedades y valores.
  • Colores y fuentes: propiedades como color, background-color, font-family, font-size.
  • Estilos de texto: text-align, text-decoration, font-weight.
  • Cajas y espaciado: margin, padding, border.
  • Comentarios: cómo documentar el código con /* comentario */.

CSS Avanzado

Durante esta exposicion profundizamos en los conceptos avanzados de CSS para crear sitios web más dinámicos, interactivos y adaptables a diferentes dispositivos.

Contenido Aprendido

  • Pseudoclases y pseudoelementos: como :hover, :first-child, ::before, etc.
  • Posicionamiento: uso de position: relative, absolute, fixed, sticky.
  • Display y Flexbox: propiedades como display: flex, justify-content, align-items.
  • Animaciones y transiciones: usando transition, keyframes, animation.
  • Media Queries: para diseño responsive.

Bootstrap

Durante esta exposicion aprendimos a utilizar Bootstrap para crear diseños responsivos y profesionales de manera eficiente, aprovechando sus componentes y utilidades.

Contenido Aprendido

  • Sistema de rejilla (grid): diseño flexible mediante filas y columnas.
  • Componentes: botones, alertas, cards, formularios, modals, etc.
  • Utilidades: clases para márgenes, paddings, colores y alineaciones.
  • Responsive design: adaptabilidad a diferentes dispositivos.
  • Extensión con JavaScript: uso de funcionalidades dinámicas como carruseles y desplegables.

Tailwind CSS

Durante esta exposicion exploramos el uso de Tailwind CSS, un framework CSS basado en utilidades que permite diseñar páginas web rápidamente con clases predefinidas, facilitando la creación de interfaces altamente personalizables y responsivas.

Contenido Aprendido

  • Clases utilitarias: control preciso de márgenes, padding, colores, tipografía y más.
  • Componentes responsivos: uso de breakpoints para adaptar diseños a diferentes dispositivos.
  • Personalización: configuración del archivo tailwind.config.js para extender temas y colores.
  • Estado y pseudoclases: clases para :hover, :focus, y otros estados interactivos.
  • Integración con frameworks: fácil uso con React, Vue, Angular, etc.

¿Qué Aprendí?

Durante este recorrido formativo por el desarrollo web, he consolidado conocimientos fundamentales y avanzados que forman la base para construir sitios web modernos, funcionales y estéticamente agradables. En la parte inicial, comprendí la estructura básica de un documento HTML, su semántica, etiquetas principales y cómo organizar el contenido de manera clara y accesible para los usuarios y motores de búsqueda. Esto me permitió establecer las bases para todo desarrollo posterior.

Posteriormente, avanzando hacia HTML avanzado, descubrí cómo utilizar etiquetas y atributos más complejos que mejoran la interactividad y la experiencia del usuario, tales como formularios, elementos multimedia y técnicas para optimizar la accesibilidad. También aprendí a estructurar páginas con mayor precisión y control.

En cuanto a CSS, tanto básico como avanzado, adquirí habilidades para aplicar estilos visuales que hacen que los sitios sean atractivos y coherentes. Con CSS básico, aprendí a controlar colores, fuentes, márgenes y formatos simples. El estudio de CSS avanzado me permitió explorar técnicas más sofisticadas como pseudoclases, flexbox, posicionamiento complejo, animaciones y media queries para diseño responsivo. Esto hace posible que los sitios web se adapten correctamente a diferentes dispositivos y tamaños de pantalla.

Además, el aprendizaje de frameworks como Bootstrap y Tailwind CSS me dio herramientas poderosas para acelerar el desarrollo y mantener una apariencia profesional con menos esfuerzo. Entendí el concepto de clases reutilizables, diseño basado en rejillas, componentes y utilidades listas para usar que facilitan la creación de interfaces limpias, responsivas y coherentes, permitiendo también personalizar estilos fácilmente.

¿Cómo Aprendí?

El aprendizaje se basó en una combinación efectiva de diferentes métodos que se complementaron entre sí. La revisión y estudio de documentación oficial y tutoriales me proporcionaron el marco teórico necesario para comprender las propiedades, etiquetas y funcionalidades.

La práctica constante mediante ejercicios guiados y la implementación de mini proyectos permitieron poner en práctica los conceptos aprendidos, experimentando directamente cómo afectan los cambios realizados y resolviendo problemas reales de desarrollo.

La exploración de ejemplos concretos, la comparación de resultados y la modificación progresiva de código me ayudaron a entender mejor el comportamiento de los elementos y estilos, así como a detectar errores y optimizar soluciones.

Además, el uso de herramientas modernas como editores de código, navegadores con herramientas de desarrollo y frameworks facilitó la experimentación y el aprendizaje interactivo, haciendo que la curva de aprendizaje fuera más dinámica y efectiva.

Finalmente, la integración de conocimientos en un portafolio y proyectos semanales reforzó la capacidad para organizar el trabajo, documentar el progreso y preparar entregables profesionales.
