SEMANA 3

Emmet

Resumen de la Semana


Se aprendió a utilizar el plugin Emmet para escribir código HTML y CSS de forma más rápida y eficiente. Se dominaron atajos de teclado en Visual Studio Code para agilizar la edición y navegación del código. Se comprendió cómo generar estructuras complejas y repetitivas en HTML usando abreviaciones con Emmet.

Emmet: Revolucionando HTML/CSS

Operadores Clave:
  • > Elementos hijos
  • + Elementos hermanos
  • ^ Subir nivel
  • * Multiplicador
  • {} Contenido de texto
  • [] Atributos
  • $ Numeración
  • . Clases
  • # IDs

Ejemplo: ul.nav>li.item$*5>a[href="#"]{Opción $}

Resultado: Lista de navegación con 5 items numerados

Atajos de Teclado Esenciales

Generales
  • Ctrl + P: Buscar archivo
  • Ctrl + Shift + P: Paleta de comandos
  • Ctrl + ,: Configuración
Edición Básica
  • Ctrl + X: Cortar línea
  • Ctrl + D: Duplicar línea
  • Alt + ↑/↓: Mover línea
Multicursor
  • Ctrl + Alt + ↑/↓: Cursor múltiple
  • Ctrl + Shift + L: Seleccionar ocurrencias
  • Alt + Click: Cursor manual

Práctica Calificada

Ejercicio 1: nav.nav>ul>li*4>a[href=page$@-1.html]{Link$@-1}

Ejercicio 2: table.table>thead>tr>th[scope="col"]{"#"}+th[scope="col"]{First}+th[scope="col"]{Second}+th[scope="col"]{Three}+tbody>(tr>th[scope="row"]{$}+td{Mark}+td{Otto}+td{Jhonson})*3

Ejercicio 3: div.container>header#header{header}+nav#nav{nav}+section#section>{section}+article#article{article}+aside#aside{aside}+footer#footer{footer}

Ejercicio 4: table>thead>tr>th{Id}+th{Nombres}+th{Apellidos}+th{Dirección}^tbody>tr*20>th[scope=row]{$}+td*3

Ejercicio 5: table>tbody>tr*20#fila$-{$}>td*2

Ejercicio 6: div.card>img[src="img.jpg" alt="Imagen del Producto"]+div.card-body>h5.card-title{Titulo del Producto}+p.card-text{Descripcion del.}+a.btn.btn-primary[href="#"]{Buy Now}

¿Qué Aprendí?

Durante esta semana, adquirí un dominio fundamental de Visual Studio Code, comprendiendo su interfaz, configuración y las poderosas herramientas que ofrece para el desarrollo web. Aprendí a utilizar la paleta de comandos para acceder rápidamente a funciones, el terminal integrado para ejecutar comandos y los atajos de teclado esenciales que optimizan el flujo de trabajo. Además, descubrí el potencial de Emmet para generar código HTML y CSS de manera eficiente mediante abreviaciones, lo que reduce significativamente el tiempo de desarrollo.

¿Cómo Aprendí?

El aprendizaje se dio mediante una combinación de material teórico, demostraciones prácticas y ejercicios guiados. Comencé instalando y configurando VSCode según las recomendaciones, luego exploré cada funcionalidad a través de ejemplos concretos. Para Emmet, practiqué con plantillas HTML complejas que luego simplifiqué usando las abreviaciones. Los ejercicios de laboratorio me permitieron aplicar estos conocimientos en proyectos reales, consolidando lo aprendido mediante la repetición y experimentación.
