SEMANA 7

Hooks en React

Resumen de la Semana


Los Hooks en React son una característica introducida en la versión 16.8 que permiten usar estado y otras funcionalidades de React sin escribir clases. Facilitan la reutilización de lógica entre componentes y hacen que el código sea más limpio y fácil de mantener. Entre los Hooks más utilizados están useState para manejar estados locales, useEffect para ejecutar efectos secundarios como llamadas a APIs, y useContext para acceder al contexto global. Gracias a los Hooks, los componentes funcionales en React pueden ser igual de poderosos que los basados en clases, pero con una sintaxis más sencilla e intuitiva.

Hooks

Introducción a los Hooks

Los Hooks son una de las características más importantes de React, introducidos en la versión 16.8. Permiten usar estado, efectos secundarios, contexto y otras funcionalidades de React en componentes funcionales, eliminando la necesidad de clases. Esto mejora la legibilidad, la modularidad y la reutilización de la lógica de componentes.

Hooks más Comunes

React proporciona varios Hooks integrados que cubren las necesidades más comunes. Aquí están algunos de los más utilizados:

useState()

Permite agregar estado local a los componentes funcionales.

const [count, setCount] = useState(0);
useEffect()

Gestiona efectos secundarios como llamadas a APIs o manipulación del DOM.

useEffect(() => {
                            // código de efecto
                            }, []);
useContext()

Accede a valores del contexto global sin necesidad de props.

const value = useContext(MiContexto);
useRef()

Referencia elementos del DOM o valores persistentes sin re-renderizar.

const inputRef = useRef(null);
useReducer()

Alternativa a useState para lógica de estado más compleja.

const [state, dispatch] = useReducer(reducer, initialState);

Combinación de Hooks

Una de las ventajas de los Hooks es que pueden combinarse para construir lógica más sofisticada. Se pueden encapsular en custom hooks reutilizables que abstraen comportamientos comunes entre componentes.

Custom Hooks

Los Custom Hooks permiten extraer lógica de componentes en funciones reutilizables. Siguen la convención de nombrarse con el prefijo use y pueden usar otros hooks internos para componer comportamientos complejos.

function useContador(inicial) {
                            const [valor, setValor] = useState(inicial);
                            const incrementar = () => setValor(v => v + 1);
                            return [valor, incrementar];
                            }
